Instagram bot for liking, commenting, and following with the ability to manage multiple accounts
Install and run
1) Install the appropriate web driver for your browser.
2) Install dependencies : '''bash pip install -r requirements.text
3) Put the username and password of your Instagram accounts in the insta_project.py file in the users section.
4) For initial login, please log in manually. Then your cookie will be saved and you will not need to log in manually the next time.
5) Remove any like, comment, or follow functions you want to run from the comment.
6) Enter the address of the desired page in each function and finally press Enter in the terminal to use the next account.
